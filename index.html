1. Introduction

The Simple Shiny App is a Shiny App published in shinyapps.io.

The application collects some information about the user, does some processing and presents a multi bar chart filtered by the data provided by the user.

The application is accessible at: https://jean-rodrigues.shinyapps.io/simpleShinyApp/

2. About the Application

The application makes use of the dataset named HairEyeColor, avaiable with R.

When the user provides its hair color and sex information, the application exhibits some info about the user and its age and presents a chart filtering the dataset by the sex and hair data provided.
